# Perfume_layering
Консольный ассистент для лееринга парфюмов на основе личных экспериментов и базы Fragrantica
### Особенности
- База из **70 000+ ароматов** (данные Fragrantica)
- Поиск по названию, аккордам, рейтингу
- **5 готовых пресетов** из моих личных экспериментов с реальными отзывами и ощущениями
- Автоматический анализ совместимости, вайба, рисков и советов по нанесению
- Красивый интерфейс в консоли (библиотека rich)
- Сохранение результатов в файл

### Как запустить
```bash
pip install pandas rich
python layering_app.py
